O que são RGB e CMYK?

RGB (Red, Green, Blue) e CMYK (Ciano, Magenta, Amarelo, Preto) são dois modelos de cores amplamente usados em diferentes contextos. RGB é utilizado para telas digitais, enquanto CMYK é usado em impressões.

Modelo RGB - Red, Green, Blue

O modelo RGB é utilizado para telas, como monitores de computador, smartphones e televisores. As cores são criadas a partir da mistura de três cores primárias: Vermelho (Red), Verde (Green) e Azul (Blue).

Quanto mais forte for cada cor primária, mais intensa a cor resultante será. Por exemplo, rgb(255, 0, 0) cria um vermelho intenso, enquanto rgb(0, 0, 255) gera um azul.

Modelo CMYK - Ciano, Magenta, Amarelo, Preto

O modelo CMYK é utilizado em impressões. As cores são criadas a partir de quatro tintas: Ciano (C), Magenta (M), Amarelo (Y) e Preto (K). Ele funciona subtraindo cores da luz branca.

Quando você mistura o ciano, magenta e amarelo, as cores se combinam para formar outras cores. O preto é usado para adicionar profundidade e contraste à impressão.

Comparando RGB e CMYK

Embora ambos sejam usados para criar cores, a principal diferença entre eles é o contexto em que são aplicados:

  • RGB: Usado para dispositivos que emitem luz (telas de computador, TVs, câmeras digitais).
  • CMYK: Usado em impressoras que trabalham com pigmentos e tintas para gerar cores.

Conclusão

Em resumo, o modelo RGB é ideal para telas e dispositivos que emitem luz, enquanto o CMYK é mais adequado para a impressão. Ambos são fundamentais no design gráfico, pois você pode precisar usar um ou outro dependendo da mídia final (digital ou impressa).

Modelo Aditivo de Cores

O modelo aditivo de cores explica como as cores são criadas pela combinação de luzes de diferentes cores primárias. No modelo aditivo, as três cores primárias são o vermelho, verde e azul (RGB). A principal ideia é que, ao combinar essas cores com diferentes intensidades de luz, podemos criar uma ampla gama de cores.

Intensidade da Luz

O modelo aditivo se baseia no princípio de que a luz é composta de diferentes comprimentos de onda. A intensidade da luz de cada cor primária pode ser ajustada para influenciar o resultado da cor combinada. Quando a luz dessas cores é combinada em diferentes intensidades, novas cores são geradas.

Combinação das Cores Primárias

Veja como as três cores primárias (vermelho, verde e azul) se combinam para formar outras cores:

Exemplos de Combinação de Cores:

Vermelho + Verde = Amarelo

Verde + Azul = Ciano

Azul + Vermelho = Magenta

Vermelho + Verde + Azul (máxima intensidade) = Branco

Sem luz (intensidade mínima) = Preto

Como a Intensidade da Luz Influencia a Cor

Se a intensidade da luz de uma das cores primárias é aumentada, a cor resultante tende a ser mais próxima da cor primária que tem maior intensidade. Por exemplo, se a intensidade da luz vermelha for aumentada, a cor resultante será mais próxima do vermelho. O mesmo vale para o verde e o azul.

Este modelo é amplamente utilizado em dispositivos como telas de computadores, televisores e projetores, onde as cores são criadas pela combinação de luzes vermelha, verde e azul.
